first site

sometime in late 2023, I decided to finally put my web dev skills to some use by compiling a bunch of crappy html games I found on Github on a website that was initially for my use only, but quickly spread around my school too. when I had to name the site, I called it "your mom gaming". as I got better with HTML and CSS, it got way better, and today, it makes use of Javascript, and soon Bootstrap.



more history

shortly after ymg was concieved, I made a similarly terrible site for my notes in my web dev class. then came a homepage to compile my sites. then I got bored with them and started throwing sites together whenever I could. that led to sites like interactions, ANGER, x, and javerscrips.
