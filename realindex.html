first site

at some point in time, I decided to finally put my web dev skills to some use by compiling a bunch of crappy html games I found on Github on a website that I could play since every other game site is blocked, and I have too much time on my hands. when I had to name the site, I called it "your mom gaming", and it now uses the 3 main front-end languages after only using html for the longest time.



more history

shortly after ymg was concieved, I made a similarly terrible site for my notes in my web dev class. then came a homepage to compile my sites. then I got bored with them and started throwing sites together whenever I could. that led to sites like your mom audio, ANGER, x, and javerscrips.
