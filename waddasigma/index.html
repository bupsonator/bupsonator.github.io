The Desktop

WaddaSigma will feature a minimalist Gnome desktop without all the clutter that some distros bring to it. Gone is the Ubuntu dock and the app menus. The authentic Gnome experience is great, so you should get to use it. The menu bar is rounded, bringing a light look to the desktop. The bar will become square again when an app is fullscreened. The bar is also transluscent, adding a custom flair to your computer.


The Apps

WaddaSigma has a selection of useful apps that most Linux users end up installing, even though most distros don't include them. Default media apps are kept to a minimum while also bringing a rich experience so you don't have to go hunting for apps that work. There are also gonna be shortcuts configured like Windows, such as Ctrl+Alt+Esc for a task manager-like app.



What will WaddaSigma be?

WaddaSigma will be a Linux distribution based off of Ubuntu 24.10. It will feature the Gnome desktop environment. It will be configured with what are in my opinion better default apps for media, such as VLC instead of RhythmBox and Gnome Videos. It will have a selection of other installed apps since Ubuntu's apps are not great for most people. It will have a more clean-looking icon pack pre-installed as well. Snapd will be removed because nobody wants that. The last major change from vanilla Ubuntu will be a customized Gnome desktop using pre-configured extensions. Oh, and it has a funny name.

Note: As of now, I don't even know if Ubuntu will work as a base since I don't have the best understanding of how Linux works. I already tried Fedora, but it went terribly. If anything goes wrong with Ubuntu, I'll probably try Debian next since the tools I have available have been tested with Debian. If that goes south, I'll try OpenSUSE. I refuse to use Arch as a base, so if none of the above work, WaddaSigma will not be released.
